什么是Apache

Apache HTTP Server是一种Web服务器，负责回应收到的web请求。

例如，Apache收到/index.php的请求时，通过CGI调用PHP解释程序，执行/index.php，返回结果给客户端

httpd是Apache超文本传输协议(HTTP)服务器的主程序。被设计为一个独立运行的后台进程，它会建立一个处理请求的子进程或线程的池。
